# Dashboard - Concept

**Dashboard** is a project designed to help you **never get lost in your tasks again**.  
It allows you to create, connect, and manage “tablets” for your tasks in a visual, intuitive way.

---

## Features / How to Use

Right-click on the dashboard to:

- Create a tablet
- Edit a tablet
- Delete a tablet
- Add a connection
- Remove connections

_Everything happens visually — just right-click and manage your workflow!_
